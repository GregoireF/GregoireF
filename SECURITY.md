# Security Policy

## Scope

This is a GitHub profile repository. No software is distributed or deployed to end users.

## Workflow security

All GitHub Actions are pinned to full commit SHAs and updated automatically via Dependabot.
Workflows follow least-privilege principles — each job declares only the permissions it requires.

## Reporting a vulnerability

If you spot a security issue in the CI/CD pipelines or workflow configuration, please open a [GitHub issue](https://github.com/GregoireF/GregoireF/issues) or contact me directly at [gregoire.favreau@ik.me](mailto:gregoire.favreau@ik.me).
